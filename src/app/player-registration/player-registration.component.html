info File Upload Guidelines

  • CSV File: Ensure the file is in CSV format with comma-separated values. The first row should contain headers such as Username, Game Name.
  • JSON File: Ensure the file is in JSON format with an array of player objects. Each object should include plUname and gameName.
  • Both CSV and JSON files must not exceed 5MB in size.
  • The file should contain only valid data (no empty or invalid entries). Invalid entries will not be saved.
  • If you encounter an error during upload, please verify the file format and ensure all fields are filled out correctly.
